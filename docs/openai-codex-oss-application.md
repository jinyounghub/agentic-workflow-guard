# Retired Internal Notes

Private grant, sponsorship, or benefit-application notes are intentionally not tracked in this public repository.

This placeholder exists only because the original public notes should be removed in a local maintenance commit. Public project documentation should focus on the tool itself: threat model, rule catalog, responsible use, adoption guide, release process, and real project metrics.

Maintainers should keep private application material in a local-only folder or a private repository outside `agentic-workflow-guard`.
