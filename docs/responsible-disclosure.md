# Responsible Disclosure

This project is for defensive workflow analysis.

## Allowed use

- Scan repositories you own or have permission to assess.
- Use synthetic vulnerable workflows for demos, tests, and documentation.
- Report findings privately to maintainers through their published security policy.
- Share generalized lessons without naming affected projects unless the maintainer has approved disclosure.

## Disallowed use

- Mass scanning public repositories and opening unsolicited public issues.
- Generating exploit payloads.
- Attempting secret exfiltration.
- Publishing real vulnerable workflow details without maintainer consent.
- Claiming adoption, stars, downloads, or security impact that does not exist.

## Reporting a vulnerability in this project

Follow [SECURITY.md](../SECURITY.md). Include a minimal synthetic workflow when possible.
